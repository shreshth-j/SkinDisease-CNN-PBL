CNN Based Skin Disease Detection

Deep learning based automatic classification of dermoscopic skin lesion images using Convolutional Neural Networks.

Problem Statement

Manual diagnosis of skin diseases requires expert dermatologists and can be subjective. This project develops an automated CNN-based classifier for early detection of skin lesions to assist medical professionals.

Literature Review

Existing CNN models classify dermoscopic images but often suffer from limited datasets, imbalance, and overfitting.

Research Gap / Innovation

Our approach combines preprocessing, augmentation, and CNN tuning on Mendeley + HAM10000 datasets to improve generalization.

System Methodology

Dataset

Mendeley Skin Dataset + HAM10000 dermoscopic images (resized, normalized)

Model

CNN using Convolution, MaxPooling, Dropout and Dense layers implemented in TensorFlow/Keras.
