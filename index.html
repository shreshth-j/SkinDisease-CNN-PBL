CNN Skin Disease Detection

Deep Learning model for automatic classification of dermoscopic skin lesions using CNN architecture.

Problem Statement

Dermatological diagnosis requires specialist knowledge and manual inspection. This project builds a CNN-based automated system for classification of skin diseases into six categories using dermoscopic images.

Literature Review

Recent studies show CNN models outperform classical machine learning in skin lesion classification.

Innovation

Optimized CNN architecture trained first on Mendeley dataset, scalable to HAM10000 for improved accuracy.

System Architecture

Dataset

3230 dermoscopic images resized to 128×128. Future training on HAM10000.

CNN Layers

Conv2D → ReLU → MaxPool → Dropout → Dense → Softmax.
