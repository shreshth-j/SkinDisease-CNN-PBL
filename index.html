CNN Skin Disease Detection

AI-based CNN model for automatic classification of dermoscopic skin lesions into multiple disease classes.

Problem Statement

Manual dermatology diagnosis requires specialist expertise and is time-consuming. Our project proposes a CNN-based automated classification system capable of identifying multiple skin diseases from dermoscopic images.

Literature Review

Existing solutions rely on traditional feature extraction and require large datasets.

Research Gap

Need optimized CNN trained on smaller dataset with scalability toward HAM10000.

System Methodology

Dataset

3230 dermoscopic images (Mendeley dataset). Future integration with HAM10000.

Model

CNN architecture with Conv2D, MaxPooling, Dropout, Dense layers using TensorFlow.
