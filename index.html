CNN SKIN DISEASE DETECTION

Deep learning classification of dermoscopic lesions using CNN

Problem Statement

Manual diagnosis of skin diseases requires dermatology expertise and may be subjective. This project develops a CNN-based automated classifier for dermoscopic lesion detection.

LITERATURE REVIEW

Existing ML systems classify skin lesions using CNN architectures but often suffer from dataset imbalance and overfitting.

RESEARCH GAP

Our system improves generalization using preprocessing, augmentation, and CNN tuning on medical dermoscopy datasets.

SYSTEM METHODOLOGY

DATASET

Mendeley + HAM10000 dermoscopic images

MODEL

CNN with Conv, Pooling, Dropout, Dense
