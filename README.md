# SkinDisease-CNN-PBL
# CNN-Based Skin Disease Detection

This project implements a Convolutional Neural Network (CNN) to classify dermoscopic images into multiple skin disease categories.

## Dataset
Mendeley Skin Disease Dataset (3230 images)

## Technologies Used
- Python
- TensorFlow / Keras
- OpenCV
- NumPy
- Pandas
- HTML/CSS/JS for visualization

## Results
Current model accuracy: ~46%

## Future Work
- Transfer learning (EfficientNet)
- Data augmentation
- Class balancing
